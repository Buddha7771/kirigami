# Kirigami
RNA secondary structure prediction via deep learning.

## Requirements
* python (>=3.8)
* C++17 compatible compiler
* virtualenv

## Install
```
git clone https://github.com/marc-harary/kirigami.git
cd kirigami
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install .
```
